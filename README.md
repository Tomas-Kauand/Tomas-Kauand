## 🚀 Proyecto Destacado: MiViandita (Tesis)

Mi enfoque principal en los últimos meses ha sido el desarrollo de un proyecto de tesis grupal, **MiViandita**, una aplicación web responsive completa desarrollada desde cero.

* **Frontend:** Angular (SPA, diseño de vistas, consumo de APIs, manejo de estado).
* **Backend:** Java con Spring Boot (API RESTful, JWT, Paginación con HATEOAS, arquitectura robusta, manejo de excepciones, lógica de negocio).
* **Consumo de APIs externas:** Google Auth, SMTP/Gmail, Cloudinary, OpenAI y GeoRef.
* **Estado del código:** Actualmente el repositorio principal se encuentra privado, ya que estamos realizando modificaciones para que MiViandita esté deployeado y accesible a través de Vercel (pronto).
